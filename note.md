# 运行
0. pnpm install 进行测试
1. pnpm dev 边测试边用
2. pnpm build 打包
3. pnpm package 生成zip包


- 就是少弄了快捷键，因为和 onCommand 相对应的，另外也少弄了content_security_policy
- build 会进行覆盖、dev 的时候不会覆盖
